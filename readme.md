# The Generalised Kernel Covariance Measure (CLeaR 2026)

This repository contains the code to reproduce the results from the simulation studies presented in the paper [The Generalised Kernel Covariance Measure (Bergen et al., 2026)](https://arxiv.org/abs/2604.03721).

To start, open `GKCM.Rproj`, which will help to determine the correct paths to use in the scripts. Afterwards load (or produce) the results of a specific simulation by running the `run_simulations.R` script from a folder. The plots in the papers can be reproduced by the accompanying `make_plots.R` script. The scripts for the simulations in the main paper are contained in the root. The script for the simulations in the appendix are contained in `appx`.
